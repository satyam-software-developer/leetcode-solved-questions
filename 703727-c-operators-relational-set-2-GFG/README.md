# [C++ Operators (Relational) | Set 2](https://www.geeksforgeeks.org/problems/c-operators-relational-set-21407/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given two numbers A and B and task is to check whether A is greater than or less than or equals to B.

&nbsp;

Example 1:

Input:
A = 2, B = 5
Output:
2 is less than 5
Explanation:
The output is self- 
explanatory.

&nbsp;

Example 2:

Input:
A = 6, B = 3
Output:
6 is greater than 3
Explanation:
The output is self- 
explanatory.

&nbsp;

Your Task:

You don't need to read input or print anything. Your task is to complete the function compareNum() which takes two integers A&nbsp;and B, if

If A &gt; B then print "A is greater than B" without quotes
If A &lt; B then print "A is less than B" without quotes
if A = B then print "A is equals to B" without quotes.

&nbsp;

Expected Time Complexity: O(1)
Expected Auxiliary Space: O(1)


Constraints:
1&lt;=A&lt;=10000
1&lt;=B&lt;=10000
&nbsp;
