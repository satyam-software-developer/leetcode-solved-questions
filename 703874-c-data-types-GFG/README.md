# [C++ Data Types](https://www.geeksforgeeks.org/problems/c-data-types1523/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Read a value and store it in the appropriate C++ Data Type.&nbsp;

Example 1:

Input: 
2 h 2.555
Output:
2
h
2.555 
Explanation:
The three inputs are printed in order.


&nbsp;

Your Task:
Your task is to complete each of the given functions&nbsp;
cppIntType() : read an integer input, store it in appropriate data type and return it.&nbsp;
cppCharType() :&nbsp;read a character&nbsp;input, store it in appropriate data type and return it.&nbsp;&nbsp;
cppFloatType() :&nbsp;read a float&nbsp;input, store it in appropriate data type and return it.&nbsp;


Expected Time Complexity: O(1)
Expected Auxiliary Space: O(1)
