# [Input Boolean Value](https://www.geeksforgeeks.org/problems/input-boolean-value/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Take a boolean value a as an input. The program should display its numeric representation:

1 for "true"
0 for "false"

Examples:&nbsp;
Input: a = true
Output: 1
Input: a = false
Output: 0
