# [Get smaller elements](https://www.geeksforgeeks.org/problems/get-smaller-elements/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
Given an array arr, return all the elements which are less than a given value x. The order of returned smaller elements should be according their appearance in the input array.&nbsp;
Examples:
Input: arr[] = [54, 43, 2, 1, 5], x = 7
Output: 2 1 5
Explanation: 2, 1, 5 are smaller than 7.
Input: arr[] = [324, 5, 2, 2, 3], x = 3
Output: 2 2
Explanation: 2, 2 are smaller than 3.
Constraints:1 &nbsp;&lt;= &nbsp;arr.length &nbsp;&lt;= &nbsp;105
