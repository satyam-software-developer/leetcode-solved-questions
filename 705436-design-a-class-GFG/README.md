# [Design a class](https://www.geeksforgeeks.org/problems/design-a-class/1?page=3&category=CPP&sortBy=submissions)
## Easy
Design a class MyClass with a public function display() that prints "Hello World". The function takes no parameters and returns void.
Note:&nbsp;

The driver's code will call the display function of the MyClass.
There is no input for this problem.

Example:
Input: None
Output: "Hello World"
