# [Express as Consecutive Number Sum](https://www.geeksforgeeks.org/problems/consecutive-numbers-for-sum3132/1)
## Easy
Given a number n, find whether n can be expressed as sum of two or more consecutive positive numbers. 
Examples :
Input: n = 10
Output: trueExplanation: 10 can be expressesd as 1 + 2 + 3 + 4.
Input: n = 8
Output: falseExplanation: 8 can't be expressesd sum of two or more consecutive numbers.
Input: n = 24
Output: trueExplanation: 24 can be expressesd as 7 + 8 + 9.
Constraints:1 ≤ n ≤ 109
