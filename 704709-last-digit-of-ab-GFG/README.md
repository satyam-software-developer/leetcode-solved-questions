# [Last Digit of a^b](https://www.geeksforgeeks.org/problems/find-last-digit-of-ab-for-large-numbers1936/1)
## Medium
Given two integers a and b in the form of strings. Return the last digit of ab.
Examples:
Input: a = "3", b = "10"
Output: 9
Explanation: 310 = 59049. Last digit is 9.
Input: a = "6", b = "2"
Output: 6
Explanation: 62 = 36. Last digit is 6.
Constraints:1&nbsp;≤ a.size(), b.size()&nbsp;≤&nbsp;1000a and b consist only of numeric digits ('0' - '9')a and b do not contain any leading zeros, except when number itself is "0"
