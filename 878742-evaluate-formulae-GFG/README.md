# [Evaluate Formulae](https://www.geeksforgeeks.org/problems/evaluate-formulae--133849/1?page=3&category=CPP&sortBy=submissions)
## Easy
Given four&nbsp;inputs that are stored in variables a, b, c, and d. You need to write an expression to evaluate the following formula. Use integer division. The expression should be&nbsp;a single statement.

Examples:
Input: a = 10, b = 4, c = 7, d = 9
Output: 11
Explanation: 10 + 4 = 14, 14 // 7 = 2(Python) or 14 / 7 = 2(Java or CPP or C), 2 + 9 = 11.
Input: a = 5, b = 6, c = 8, d = 9Output: 10Explanation: 5 + 6 = 11, 11 // 8 = 1(Python) or 11 / 7 = 1(Java or CPP or C), 1 + 9 = 10.
Constraints:1 ≤ a, b, c, d ≤ 100
