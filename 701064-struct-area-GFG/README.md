# [Struct area](https://www.geeksforgeeks.org/problems/struct-area/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
As now you have learnt about how to create structures. Let us now see how to assign values to data members of structures and perform some operation.
Given a structure area with side, len and wid as data members. Your task is to assign values to data members and find area of square with side side&nbsp;and area of rectangle with length len&nbsp;and width wid. You will be given a function find_area(side, len, wid) for side, len and wid as an argument.
Example :  
Input: side = 25, len = 40, wid = 60
Output: 625 2400
Explanation: Area of square with given side 25 is: 625(side*side) and area of rectangle is: 2400(len*wid).
