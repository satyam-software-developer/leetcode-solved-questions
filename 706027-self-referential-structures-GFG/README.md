Self Referential structures are those structures that have one or more pointers which point to the same type of structure, as their member. Structures are important concept used during implementing Linked List, Trees and other data structures. Let's start with creating struct.
Your have&nbsp;to create Linked List struct node. In the Linked List you should have two members one is data and the other is pointer which points to next node: int data and struct node *next
