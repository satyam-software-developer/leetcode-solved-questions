# [Print Diagonally](https://www.geeksforgeeks.org/problems/print-diagonally4331/1)
## Easy
Give a n * n square matrix mat[][], return all the elements of its anti-diagonals from top to bottom. 
Examples :
Input: n = 2, mat[][] = [[1, 2],
                        [3, 4]]
Output: [1, 2, 3, 4]
Explanation:

                       [7, 8, 9]]
