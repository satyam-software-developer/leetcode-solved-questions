# [AP Term](https://www.geeksforgeeks.org/problems/ap-term--150919/1?page=2&category=CPP&sortBy=submissions)
## Easy
Given three integers a, d and n.&nbsp;Where a is the first term, d is the common difference of an A.P.&nbsp;&nbsp;Calculate the nth&nbsp;term of A.P.&nbsp; The nth term is given by&nbsp;an&nbsp;=&nbsp;a + (n-1)d 
Examples:&nbsp;
Input: a = 5, d = 2, n = 5
Output: 13
Explanation: anth = a + (n-1)d = 5 + (5-1)*2 = 5 + 8 = 13
Input: a = 10, d = 10, n = 101 
Output: 1010 
Explanation: anth = a + (n-1)d = 10 + (101-1)*10 = 10 + 1000 = 1010.
