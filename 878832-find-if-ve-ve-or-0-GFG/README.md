# [Find if +ve, -ve or 0](https://www.geeksforgeeks.org/problems/find-if-ve-ve-or-0--111305/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
You are given a number a. Your task is to print "Positive" if the number is positive, print "Negative" if the number is negative, and print "Zero" if it is 0.
Examples :
Input: a = 4
Output: Positive
Input: a = -2
Output: Negative
Input: a = 0
Output: Zero
Constraints:-105&nbsp;≤ a ≤ 105&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;
