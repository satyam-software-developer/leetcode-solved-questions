# [Java Streams Max](https://www.geeksforgeeks.org/problems/java-streams-max/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
Given an integer array arr[], return the maximum element from the array using streams.
Examples:
Input: arr[] = [1, 2, 3, 4]
Output: 4
Explanation: From the given array, 4 is max.
Input: arr[] = [3, 2, 1]
Output: 3
Explanation: From the given array, 3 is max.
Constraints:1&nbsp;≤&nbsp;arr.length&nbsp;≤ 1031&nbsp;≤&nbsp;arr[i]&nbsp;≤ 103
