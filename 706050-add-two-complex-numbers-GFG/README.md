# [Add two complex numbers](https://www.geeksforgeeks.org/problems/add-two-complex-numbers/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given two complex numbers c1 and c2, add them and return the resulting complex number.
A structure Complex is already defined with the following data members:

real: stores the real part of the complex number.
img: stores the imaginary part of the complex number.

Example :
Input: c1.real = 3, c1.img = 6, c2.real = -1, c2.img = 4
Output: 2 + 10iExplanation: The real part of the result is 3 + (-1) = 2. The imaginary part of the result is 6 + 4 = 10. Therefore, the resulting complex number is 2 + 10i.
