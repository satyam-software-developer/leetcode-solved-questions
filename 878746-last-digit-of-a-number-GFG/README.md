# [Last Digit of a number](https://www.geeksforgeeks.org/problems/last-digit-of-a-number--145429/1?page=1&category=CPP&sortBy=submissions)
## Easy
Given an integer n.&nbsp;Write a program to return the last digit of the number.
Examples:
Input: n = 10
Output: 0
Input: n = 9768
Output: 8

Constraints:
-109 ≤ n ≤ 109
