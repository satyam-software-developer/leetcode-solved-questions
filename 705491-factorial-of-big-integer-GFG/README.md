# [Factorial of BigInteger](https://www.geeksforgeeks.org/problems/factorial-of-biginteger/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
Given one integer n. Your task is to find factorial of n. You have to return value of factorial n in BigInteger.
Example:
Input: n = 10 
Output: 3628800
Explanation: Factorial of n is 3628800.
Input: n = 5Output: 120Explanation: Factorial of n is 120.
Constraints:&nbsp;1 &nbsp;&lt;= &nbsp;n &nbsp;&lt;= &nbsp;100
