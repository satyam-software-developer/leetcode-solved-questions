# [Print Statement](https://www.geeksforgeeks.org/problems/print-statement--112959/1?page=3&category=CPP&sortBy=submissions)
## Easy
Given an input String s, you need to take the string at input and print it then move the cursor to the next line.
Examples:
Input: s = "Hello World"Output: Hello World Explanation: The input is "Hello World", and we just print it.
