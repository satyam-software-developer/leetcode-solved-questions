# [Sort in Descending](https://www.geeksforgeeks.org/problems/sort-in-descending--141631/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given an array arr[] of integers, sort the array in descending order, driver code will print arr to verify the solution.
Examples :
Input: arr = [324, 5, 2, 2]
Output: 324 5 2 2
Input: arr = [54, 43, 2, 1, 5]
Output: 54 43 5 2 1
Constraints:1 ≤ |arr| ≤ 1030&nbsp;≤ arr[i]&nbsp;≤ 103
