# [Type Inference](https://www.geeksforgeeks.org/problems/type-inference--141631/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given a C++ program having two variables a and b, it can be of any datatype like int, long, double, etc. You have to store its type id into the&nbsp;id1&nbsp;and&nbsp;id2&nbsp;variables.
Examples:
Input: a = 5, b = 's'
Output: i cExplanation: typeid of int is "i" and typeid of char is "c".
