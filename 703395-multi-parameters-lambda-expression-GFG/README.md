# [Multi Parameters Lambda Expression](https://www.geeksforgeeks.org/problems/multi-parameters-lambda-expression/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
You need to use lambda expression to add two numbers a and b.You need to complete the function&nbsp;helperFunction&nbsp;that does not take any&nbsp;argument. This function expects an object of Add as a return&nbsp;type. Add is an interface that has the member function addParameters. Your&nbsp;helperFunction&nbsp;uses&nbsp;lambda expression&nbsp;to implement the&nbsp;addParameters&nbsp;within&nbsp;helperFunction&nbsp;itself. Then you&nbsp;return&nbsp;the object. The&nbsp;driver code&nbsp;will call the addParameters method and print the result using it.
Examples :&nbsp;&nbsp;
Input: a = 5, b = 7
Output: 12
Explanation: 5+7 = 12
Input: a = 1, b = 5
Output: 6
Explanation: 1+5 = 6

Constraints:1 &lt;= a, b &lt;= 103
