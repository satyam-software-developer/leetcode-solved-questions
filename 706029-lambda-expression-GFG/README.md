# [Lambda Expression](https://www.geeksforgeeks.org/problems/lambda-expression-in-c/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
You need to use lambda expression to calculate square of a given number n.
Examples :
Input: n = 5 
Output: 25
Input: n = 6
Output: 36

Constraints :
-104 ≤ n ≤ 104&nbsp;
