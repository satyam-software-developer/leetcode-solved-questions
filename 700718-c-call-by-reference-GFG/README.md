# [C++ Call by reference](https://www.geeksforgeeks.org/problems/c-call-by-reference/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given two positive integers a and b, reverse their digits and then swap them.
Examples :  
Input: a = 1234, b = 67890
Output: 9876 4321
Explanation: Reversing the given two numbers will give: 4321 and 9876. After swapping it would come as: 9876 and 4321.

Input: a = 10000, b = 3254
Output: 4523 1

Constraints:0 ≤ a,b ≤ 105
