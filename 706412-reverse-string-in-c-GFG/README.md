# [Reverse String in C](https://www.geeksforgeeks.org/problems/reverse-string-in-c/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given a string str, you need to reverse it.Examples:
Input: str = "Hello"Output: olleH
Explanation: Reverse of Hello is olleH

Input: str = "World"Output:dlroWExplanation: Reverse of World is dlroW

Constraints: 1 ≤ |str| ≤ 10^5
