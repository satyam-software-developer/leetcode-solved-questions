# [C++ STL List](https://www.geeksforgeeks.org/problems/c-stl-list/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
You are given an empty list. You have to implement different functions of&nbsp;List A i.e. adding an element in front and end, removing an&nbsp;element&nbsp;from the front and end, sorting elements, reversing the list and printing the list.

Example:

Input:
1
8
1 5
8 1
3
4
5
6
1 6
7
Output:
5
1 5
5 1
2
5 1
5
5 6
6


Input:
The first line of input contains an integer T denoting the no of test cases.For each&nbsp;test case, the first line of input contains an integer Q denoting the no of queries. Then in the next line are Q&nbsp;space separated queries.
A query can be of eight types&nbsp;
1&nbsp;x (Adds an element x to the list A&nbsp;at the end and print list A)
2&nbsp;(Sorts the list A in ascending order and print list A )
3 (Reverses the list A and print list A)
4&nbsp;(Prints the size of the list A)
5 (Prints space-separated values of the list, if the list is empty output&nbsp;-1)
6 (Remove an element from the back of the list and print list A )
7(Remove an element from the&nbsp;front of the list and print list A)
8 x (Adds&nbsp;element&nbsp;x in front of the list&nbsp;and print list A)

Output:
The output for each test case will be according to the query that is performed and if the list is empty output is -1.&nbsp;

Constraints:
1&lt;=T&lt;=100
1&lt;=Q&lt;=100
