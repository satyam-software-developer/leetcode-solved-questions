# [Address Operator](https://www.geeksforgeeks.org/problems/address-operator/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given a variable a, store its address in variable addr. Then, print the value of variable a using addr without newline.
Examples :
Input: a = 4 Output: 4  
Input: a = 61
Output: 61
