# [Compute a/b](https://www.geeksforgeeks.org/problems/compute-ab/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given two integers&nbsp;a&nbsp;and&nbsp;b. Find the value of&nbsp;a/b, where&nbsp;b is not equal to zero. After computation, return the value. The answer should be in double&nbsp;format with precise values.
Examples:
Input: a = 5, b = 2Output: 2.5
Input: a = 4, b = 4Output: 1.0
Constraints:-103&nbsp;≤ a, b&nbsp;≤ 103b != 0
