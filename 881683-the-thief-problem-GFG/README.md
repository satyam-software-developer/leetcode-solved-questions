# [The Thief Problem](https://www.geeksforgeeks.org/problems/the-thief-problem--180930/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
You are given an integer array arr[]. The elements of array represent the value of items. You are also given an integer k. Now, a thief wants to commit a burglary but he can only pick k items from the given items. Help him so that he gets the maximum total value out of this theft.
Examples:
Input: arr[] = [3, 7, 2, 5, 12, 30], k = 3
Output: 49
Explanation: Here k = 3. The thief should pick the bolded items: 3 7 2 5 12 30. So, the sum of these is 49.

Input: arr[] = [8, 10, 2, 50, 80, 20], k = 4
Output: 160
Explanation: Here k = 4. The thief should pick the bolded items: 8 10 2 50&nbsp; 80 20. So, the sum of these is 160.
