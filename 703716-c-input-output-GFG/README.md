# [C++ Input / Output](https://www.geeksforgeeks.org/problems/c-input-output2432/1?page=2&category=CPP&sortBy=submissions)
## Easy
Given two&nbsp;integers A and B. Your task is to&nbsp;return their product.

&nbsp;

Example 1:

Input:
A = 1, B = 2
Output:
2
Explanation:
Multiplication of 1 and 2 is 2.


&nbsp;

Example 2:

Input:
A = 8, B = 7
Output:
56
Explanation:
Multiplication of 8 and 7 is 56.

&nbsp;

Your Task:

You don't need to read input or print anything. Your task is to complete the function multiplication() which takes&nbsp;two integers a and b and returns their product.

&nbsp;

Expected Time Complexity: O(1)
Expected Auxiliary Space: O(1)

&nbsp;

Constraints:
0&lt;=a,b&lt;=10000
