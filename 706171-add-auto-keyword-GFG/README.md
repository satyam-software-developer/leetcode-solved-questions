# [Add - Auto Keyword](https://www.geeksforgeeks.org/problems/add-auto-keyword--141631/1?page=4&category=CPP&sortBy=submissions)
## Easy
Given one integer&nbsp;a,&nbsp;float&nbsp;b and&nbsp;&nbsp;long c variable. Add all these variables and store them into&nbsp;auto&nbsp;variable ans.
Examples:
Input: a= 3, b= 4.2, c= 23
Output: 30.2Explanation: a + b + c = 30.2
Input: a= 2, b= 7.42, c= 100
Output: 109.42
