# [Size of Binary Tree](https://www.geeksforgeeks.org/problems/size-of-binary-tree/1)
## Easy
Given the root of a binary tree, return the size of the tree. The size of a binary tree is the total number of nodes in the tree.
Examples:
Input:      Output:  3Explanation: There are 3 nodes in the given binary tree, so its size is 3.
Input:Output: 6Explanation: There are 6 nodes in the given binary tree, so its size is 6.
Constraints:1 ≤ number of nodes ≤ 1051 ≤ node-&gt;data ≤ 105
