# [Constructor Overloading](https://www.geeksforgeeks.org/problems/constructor-overloading/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Design a class as described below.

class: Userinstance variable: name(String) constructor: First: { parameter: none, task: initialize the instance variable to "Default".}Second: { parameter: name(String), task: initialize the instance variable to the value provided in the parameter.}

Example:
Input:  "Geeks"
Output: "Default"
        "Geeks"
