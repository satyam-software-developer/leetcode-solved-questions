# [Even Odd Game](https://www.geeksforgeeks.org/problems/even-odd-game--153136/1?page=2&category=CPP&sortBy=submissions)
## Easy
Given a number n, number of apples in a bag. You and your friend are picking one apple turnwise from the bag. It is given that the first attempt is always by you. The person picking the last apple will be the winner.&nbsp;

If you will win: print "You" (without quotes)
If your friend will win: print "Friend" (without quotes)

Examples:
Input: n = 9
Output: You
Input: n = 4
Output: Friend
Constraints:1&nbsp;≤&nbsp;n ≤ 100
