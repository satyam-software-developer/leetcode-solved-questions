# [Insertion in deque](https://www.geeksforgeeks.org/problems/insertion-in-deque/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
Given an array arr[] containing non-negative integers. You need to insert all elements of the array to deque and return it.
Examples:
Input: arr[] = [1, 2, 3, 4, 5]
Output: [1, 2, 3, 4, 5]
Explanation: After insert in the deque it will look like [1, 2, 3, 4, 5].

Input: arr[] = [1]
Output: [1]
Explanation: After insert in the deque it will look like [1].
Constraints:1 ≤ arr.size() ≤ 105&nbsp; &nbsp;1&nbsp; ≤&nbsp;arr[i] ≤&nbsp;105
