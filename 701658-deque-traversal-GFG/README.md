# [Deque Traversal](https://www.geeksforgeeks.org/problems/dequeue-traversal/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
Given a deque dq containing integer elements, the task is to traverse the dq and print elements of it.&nbsp;Note: Print a newline at the end.
Examples:
Input: dq = [1, 2, 3, 4, 5]
Output: 1 2 3 4 5
Explanation: dq will look like [1, 2, 3, 4, 5].
Input: dq = [1]
Output: 1
Explanation: dq will look like [1].
