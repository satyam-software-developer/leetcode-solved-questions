# [Right Angle Triangle Pattern](https://www.geeksforgeeks.org/problems/right-angle-triangle-1605685807/1?page=1&category=CPP&sortBy=submissions)
## Easy
Given an integer n. Write a program to print the Right angle triangle wall. The length of perpendicular and base is s.&nbsp;Note: Print exactly single " " after "*". Print a new line after printing the triangle.
Example:
Input: n = 4
Output:
* * * * * * * * * * Explanation: Length of perpendicular and base of triangle is 4 .
Input: n = 3
Output:
* * * * * * Explanation: Length of perpendicular and base of triangle is 3 .
Constraints:1 ≤ n ≤ 20
