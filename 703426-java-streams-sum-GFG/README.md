## Easy
Given an integer array arr[], find the sum of the array using streams.
Examples :
Input: 1 2 3 4
Output: 10
Explanation: 1+2+3+4 = 10
Input: 3 2 1
Output: 6
Explanation: 3+2+1 = 6
