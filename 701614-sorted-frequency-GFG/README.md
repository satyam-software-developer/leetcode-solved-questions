# [Sorted Frequency](https://www.geeksforgeeks.org/problems/sorted-frequency/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
You are given an integer array arr[]. You need to print the distinct elements of array in sorted order(increasing) together with their frequency.Note: Print each element in a new line.
Example:
Input: arr[] = [3, 3, 2, 2, 1, 4]
Output:1 12 23 24 1
Explanation: 3 appears 2 times. 2 appears 2 times. 1 appears 1 time, and 4 appears 1 time. So in sorted order we write elements and their frequencies.
Constraints:1 ≤ arr.size() ≤ 104
