# [Scientific Format](https://www.geeksforgeeks.org/problems/scientific-format--141631/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
There are times when your answer is a&nbsp;double&nbsp;data type and you want to show the same in scientific format. Here, we have&nbsp;to get a&nbsp;scientific&nbsp;format answer out of a double data type. You have to input&nbsp; a and need to output its scientific format upto 4 precision.
Examples :
Input: a = 5.43 
Output: 5.4300e+00
Constraints:1 &lt;= a &lt;= 100
