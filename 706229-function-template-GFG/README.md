# [Function Template](https://www.geeksforgeeks.org/problems/function-template/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Medium
Given an array arr[] of size n. The task is to find the largest element in it. The input values of the array can be of any of two numeric data types&nbsp;int and float.
Note :- In input format you need to specify the data type of the array. It is given as an input to program by an integer variable K. When K is 1, it means the data type of the array elements is int. And when K is 2, the data type of the array elements is float.Array may contains duplicates.
Examples :
Input:&nbsp;n = 5, K = 1 arr[] = [1, 8, 7, 56, 90]Output:&nbsp;90Explanation:&nbsp;The largest element of given array is 90.
Input: n = 7, K = 2 arr[] = [1.1, 2.2, 0, 3.2, 2.7, 4.6, 5.0] Output: 5.0 Explanation: The largest element of given array is 5.
