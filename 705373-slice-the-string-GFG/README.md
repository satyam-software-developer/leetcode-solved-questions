# [Slice The String](https://www.geeksforgeeks.org/problems/slice-the-string--141628/1?page=4&category=CPP&sortBy=submissions)
## Easy
Given a string s, slice it so that the output is a substring that contains all the characters except the first and last.
Note: The length of the s will be greater than 2.&nbsp;
Examples:
Input: s = "Hello"
Output: ell
Explanation: The first and last character are ignored.
Input: s = "World"Output: orlExplanation: The first and last characters are ignored.
