# [Switch Statement](https://www.geeksforgeeks.org/problems/switch-statement/1?page=2&category=CPP&sortBy=submissions)
## Easy
Given a number n, use a switch statement to return "One" if the given number is equal to 1, "Two" if the number is 2 and so on till 9 ("Nine") else return "Unknown"(without quotes).&nbsp;Examples:
Input: n = 10
Output: Unknown
Input: n = 1
Output: One
