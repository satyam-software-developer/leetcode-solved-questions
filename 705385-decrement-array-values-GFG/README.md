# [Decrement Array Values](https://www.geeksforgeeks.org/problems/decrement-array-values/1?page=3&category=CPP&sortBy=submissions)
## Easy
You are given an array arr[] that contains integers. You need to decrement each element of the array by&nbsp;1 and return the array.
Examples:
Input: arr[] = [54, 43, 2, 1, 5]
Output: 53 42 1 0 4
Explanation: Just decrement the numbers by 1.
Input: arr[] = [324, 5, 2, 2]
Output: 323 4 1 1
Explanation: Just decrement the numbers by 1.
