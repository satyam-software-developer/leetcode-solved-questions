# [C++ Functions (Sum of numbers) | Set 1](https://www.geeksforgeeks.org/problems/c-functions-set-1introduction/1?page=3&category=CPP&sortBy=submissions)
## Easy
Given three numbers A, B, C you have to write a function named calcSum() which takes these 3 numbers as arguments and returns their sum.

Input: 
The input line contains T, which denotes the number of testcases. Then T test cases follow. Each test case consists of a single line which&nbsp;contains three space separated integers A, B, and C.

Output:
Corresponding to&nbsp;each testcase, output sum of A, B and C in a new line.

User Task:
Since this is a functional problem you don't have to worry about input, you just have to complete the function calcSum().

Constraints:
1 &lt;= T &lt;= 105
1 &lt;= A &lt;= 102
1 &lt;= B &lt;= 102
1 &lt;= C &lt;= 102

Example:
Input:
3
1 2 3
5 6 7
2 5 3

Output:
6
18
10

Explanation:
Testcase 1: Sum of the given 1,2 and 3 is 6.
