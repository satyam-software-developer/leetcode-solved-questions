# [One Parameter Lambda Expression](https://www.geeksforgeeks.org/problems/one-parameter-lambda-expression/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
You need to use lambda expression to multiply a given number n by 5.You need to complete the function&nbsp;helperFunction&nbsp;that does not take any&nbsp;argument. This function expects an object of Multiply as a return&nbsp;type. Multiply is an interface that has the member function multiplyBy5. Your&nbsp;helperFunction&nbsp;uses&nbsp;lambda expression&nbsp;to implement the&nbsp;multiplyBy5&nbsp;within&nbsp;helperFunction&nbsp;itself. Then you&nbsp;return&nbsp;the object. The&nbsp;driver code&nbsp;will call the multiplyBy5 method and print the result using it.
Examples:
Input: 5
Output: 25
Explanation: 5 * 5 = 25
Input: 6
Output: 30
Explanation: 6 * 5 = 30
Constraints:1 &lt;= n &lt;= 103
