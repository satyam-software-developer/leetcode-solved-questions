# [No Parameter Lambda Expression](https://www.geeksforgeeks.org/problems/no-parameter-lambda-expression/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
You need to use lambda expression to print "Hello".You need to complete the function&nbsp;helperFunction&nbsp;that does not take any&nbsp;argument. This function expects an object of Hello as a return&nbsp;type. Hello is an interface that has the member function sayHello. Your&nbsp;helperFunction&nbsp;uses&nbsp;lambda expression&nbsp;to implement the&nbsp;sayHello&nbsp;within&nbsp;helperFunction&nbsp;itself. Then you&nbsp;return&nbsp;the object. The&nbsp;driver code&nbsp;will call the sayHello method using the returned object.

Lambda Expression :- https://www.geeksforgeeks.org/lambda-expressions-java-8/

Example:
Input: No Input
Output: Hello
