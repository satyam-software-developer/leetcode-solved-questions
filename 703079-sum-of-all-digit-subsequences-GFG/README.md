# [Sum of All Digit Subsequences](https://www.geeksforgeeks.org/problems/sum-of-all-sub-sequences-of-a-number3437/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given a string s consisting of digits, find the sum of all digits present in every non-empty subsequence of s.
Examples:
Input: s = "123" 
Output: 24
Explanation: The non-empty subsequences are: [1], [2], [3], [1, 2], [1, 3], [2, 3], [1, 2, 3]. The sum of all digits in these subsequences is 24.

Input: s = "405"
Output: 36
Explanation: The non-empty subsequences are: [4], [0], [5], [4, 0], [4, 5], [0, 5], [4, 0, 5]. The sum of all digits in these subsequences is 36.

Constraints:1 ≤ |s| ≤ 24
