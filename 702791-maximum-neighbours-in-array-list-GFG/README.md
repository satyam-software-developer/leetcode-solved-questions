# [Maximum Neighbours In ArrayList](https://www.geeksforgeeks.org/problems/maximum-neighbours-in-arraylist/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
Given an integer ArrayList arr[], return the maximum for every adjacent pairs in the ArrayList.&nbsp;
Examples:
Input: [1, 2, 2, 3, 4, 5]
Output: [2, 2, 3, 4, 5]
Explanation: Maximum of arr[0] and arr[1] is 2, that of arr[1] and arr[2] is 2, ... and so on. For last two elements, maximum is 5.

Input: [5, 5]
Output: [5]
Explanation: We only have two elements so max of 5 and 5 is 5 only.
Constraints:2 ≤ arr.size()&nbsp;≤ 1061&nbsp;≤ arr[i]&nbsp;≤ 106
