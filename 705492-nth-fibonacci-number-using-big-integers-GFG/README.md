# [Nth Fibonacci number using BigIntegers](https://www.geeksforgeeks.org/problems/nth-fibonacci-number-using-bigintegers/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
Given one integer n. Your task is to find nth fibonacci number. You have to return value of nth fibonacci number in BigInteger. 

Example: 1st Fibbonacci number is 1, 2nd is 1, 3rd is 2, 4th is 3 and so on.

Examples:
Input: n = 50
Output: 12586269025
Explanation: 50th Fibonacci number is 12586269025.

Input: n = 6Output: 8Explanation: 6th Fibonacci number is 8.
Constraints:&nbsp;1 &nbsp;&lt;= &nbsp;n &nbsp;&lt;= &nbsp;100
