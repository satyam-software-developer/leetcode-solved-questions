# [Find if +ve, -ve or 0](https://www.geeksforgeeks.org/problems/find-if-ve-ve-or-0--141631/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
You will be given a number a. Your task is to print "Positive" if the number is positive, print "Negative" if negative and print "Zero" if it is 0 (without adding a line break statement).
Examples :
Input: a = 4
Output: Positive
Input: a = -2
Output: Negative
Constraints:-105 ≤ a ≤ 105&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;
