# [Check Consonants and Vowels](https://www.geeksforgeeks.org/problems/consonants-and-vowels-check-java/1?page=2&category=CPP&sortBy=submissions)
## Easy
You are given a string s, containing only lowecase letters. Count the number of vowels and the number of consonants. 

If vowel count &gt; consonant count then print - “Yes”(without quotes).
If vowel count &lt; consonant count then print - “No”(without quotes).
If vowel count = consonant count then print - “Same”(without quotes).

Examples:
Input: s = thequickbrownfoxjumpsoverthelazydog
Output: No
Input: s = aaaaaa
Output: Yes
Input: s = ababOutput: Same
Constraints:1 ≤ |s| ≤ 100Where, |s| is the size of string.
