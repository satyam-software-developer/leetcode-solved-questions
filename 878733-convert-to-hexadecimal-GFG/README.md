# [Convert to Hexadecimal](https://www.geeksforgeeks.org/problems/convert-to-hexadecimal--122416/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given an integer a,&nbsp;convert the decimal number a into its hexadecimal representation and print the answer.
Examples :&nbsp;
Input: a = 26
Output: 1a
Input: a = 20
Output: 14
Constraints:1 ≤ a ≤ 100
