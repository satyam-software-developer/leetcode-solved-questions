# [Number of BST From Array](https://www.geeksforgeeks.org/problems/number-of-bst-from-array/1)
## Hard
You are given an integer array arr[] containing distinct elements.
 Your task is to return an array where the ith element denotes the number of unique BSTs formed when arr[i] is chosen as the root.
Examples :
Input: arr[] = [2, 1, 3]
Output: [1, 2, 2]Explanation: 
Input: arr[] = [2, 1]
Ouput: [1, 1]

Constraints:1 ≤ arr.size() ≤ 61 ≤ arr[i] ≤ 15
