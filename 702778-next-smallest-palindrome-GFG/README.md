# [Next Smallest Palindrome](https://www.geeksforgeeks.org/problems/next-smallest-palindrome4740/1)
## Hard
Given a number, in the form of an array num[]&nbsp;containing digits from 1 to 9(inclusive). Find the next smallest palindrome strictly larger than the given&nbsp;number.
Examples:
Input: num[] = [9, 4, 1, 8, 7, 9, 7, 8, 3, 2, 2]
Output: [9, 4, 1, 8, 8, 0, 8, 8, 1, 4, 9]
Explanation: Next smallest palindrome is 9 4 1 8 8 0 8 8 1 4 9.

Input: num[] = [2, 3, 5, 4, 5]
Output: [2, 3, 6, 3, 2]
Explanation: Next smallest palindrome is 2 3 6 3 2.

Constraints:1 ≤ n ≤ 1051 ≤ num[i] ≤ 9
