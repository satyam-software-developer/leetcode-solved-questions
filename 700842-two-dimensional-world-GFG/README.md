# [Two Dimensional World](https://www.geeksforgeeks.org/problems/two-dimensional-world/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
Given a 2D array arr[][] of size n × n containing integers. Print the elements of each row in a separate line, with a single space between consecutive elements.
Note: Do not print any extra spaces at the end of a line.
Example :  
Input: n = 3, arr[][]= [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
Output: 1 2 3
4 5 6
7 8 9
Explanation: Elements of 2D array are printed in row-column order.
Constraints:1 ≤ n ≤ 1001 ≤ arr[i][j] ≤ 106
