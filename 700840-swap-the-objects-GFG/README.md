# [Swap the objects](https://www.geeksforgeeks.org/problems/swap-the-objects/1?page=1&category=CPP&status=unsolved&sortBy=submissions)
## Easy
A class containing three attributes: name (string), number (integer), and height (integer). Given two objects of this class, swap the objects.&nbsp; &nbsp;&nbsp;
Example:
Input: obj1 = ["Morty", 1234, 6], obj2 = ["Rick", 5678, 5]Output: obj1 = ["Rick", 5678, 5], obj2 = ["Morty", 1234, 6]Explanation: Swapped values of obj1 and obj2.
Constraints:1&nbsp;≤&nbsp;|name| ≤&nbsp;1001&nbsp;≤&nbsp;number&nbsp;≤&nbsp;999991&nbsp;≤&nbsp;height&nbsp;≤&nbsp;10
